# Product Strategy

## What SaunaPlanet Is

SaunaPlanet is not a sauna directory.

SaunaPlanet is a platform connecting:

* sauna facilities
* sauna events
* sauna masters
* certifications
* reviews
* reservations
* payments

The goal is to build the leading sauna ecosystem in Europe.

---

## Core Product Loop

The primary user journey is:

User
↓
Discovers Event
↓
Discovers Sauna Master
↓
Discovers Sauna Facility
↓
Attends Event
↓
Leaves Review
↓
Returns

Every major feature should strengthen this loop.

---

## Product Differentiators

### 1. Sauna Masters

Sauna masters are first-class entities.

Users should be able to:

* browse masters
* follow masters
* review masters
* discover events by master
* view certifications

### 2. Events

Events are more important than facilities.

Users should often discover a facility because of an event.

### 3. Certifications

Certifications create trust.

The platform should support:

* qualifications
* federation certificates
* competition achievements
* rankings

### 4. Community

The platform should encourage:

* reviews
* rankings
* recommendations
* recurring participation

---

## Marketplace Direction

Long-term SaunaPlanet should evolve into a marketplace.

Marketplace components:

### Event Reservations

Users can:

* reserve places
* join waiting lists
* pay online

### Private Saunas

Private owners can:

* list saunas
* publish availability
* accept reservations
* receive payments

### Sauna Experiences

Future possibility:

* guided experiences
* workshops
* training sessions
* certification courses

Potential revenue sources:

## Architecture Guidance

When designing features:

Prefer:

* extensible schemas
* ownership models
* approval workflows
* compatibility with payments

Avoid:

* hardcoded assumptions
* directory-only features
* designs that block future reservations or marketplace functionality

* reservations

## Mobile Usage Vision

Typical user scenarios:

- discovering nearby saunas while travelling
- finding events near current location
- booking a sauna session from a phone
- uploading sauna photos directly from a camera
- following favourite sauna masters
- reserving private garden saunas

The platform should feel natural as a mobile application.
