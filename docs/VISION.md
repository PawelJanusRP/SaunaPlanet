# SaunaPlanet Vision

## Purpose

SaunaPlanet is not a sauna directory.

SaunaPlanet is a platform connecting sauna facilities, sauna events, sauna masters and sauna enthusiasts.

The goal is to become the primary sauna discovery platform in Europe.

---

# Core Concept

Most sauna websites answer:

"Where is a sauna?"

SaunaPlanet should answer:

* Which sauna should I visit?
* What events are happening there?
* Who is conducting them?
* What certifications do they have?
* How do other users rate the experience?

---

# Core Entities

## Sauna Facilities

Sauna facilities are the foundation of the platform.

Examples:

* public saunas
* sauna resorts
* spa facilities
* thermal complexes
* hotel saunas

Each facility should have:

* profile
* photos
* description
* location
* reviews
* ratings
* events
* sauna masters

---

## Sauna Events

Events are one of the most important differentiators.

Examples:

* sauna ceremonies
* themed events
* competitions
* educational workshops
* festivals

Users should be able to discover events before discovering facilities.

---

## Sauna Masters

Sauna masters are first-class entities.

Each sauna master should have:

* profile
* biography
* certifications
* ranking
* reviews
* event history
* associated facilities

Users should be able to follow and discover sauna masters.

---

## Certifications

Certifications create trust.

Examples:

* official qualifications
* federation certifications
* competition achievements
* training credentials

Certifications should be visible on both:

* sauna master profiles
* sauna facility profiles

---

## Rankings

Rankings should exist for:

* sauna facilities
* sauna events
* sauna masters

The goal is to help users identify quality experiences.

---

# User Journey

Example:

User opens SaunaPlanet.

Instead of searching for a sauna, the user discovers:

* an interesting ceremony
* conducted by a highly rated sauna master
* at a nearby sauna facility

The event becomes the entry point.

---

# Long-Term Ecosystem

Sauna Facility
↓
Events
↓
Sauna Masters
↓
Certifications
↓
Reviews
↓
Rankings

All major platform features should strengthen this ecosystem.

# Future Features

## Booking System

Users can reserve places at events.

---

## Payments

Users can pay for:

* events
* memberships
* reservations

directly through SaunaPlanet.

---

## Private Garden Saunas

Private owners can publish their saunas.

Features:

* availability calendar
* reservations
* payments
* reviews

---

## Subscription Model

Premium users may receive:

* advanced search
* event alerts
* favorite masters tracking
* personalized recommendations

---

## Verification Workflow

Facilities and sauna masters should be verifiable.

Verification may include:

* manual review
* certification checks
* trusted partner approval

# Product Philosophy

The platform should prioritize:

1. Discoverability
2. Trust
3. Community
4. Professionalism
5. High-quality sauna culture

Every major feature should support at least one of these principles.

---

# Success Definition

Success is not measured by the number of sauna facilities.

Success is measured by the quality of the ecosystem connecting:

* facilities
* events
* sauna masters
* certifications
* users

into a single platform.
